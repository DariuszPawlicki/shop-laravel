# shop-laravel

## Odpalenie aplikacji:

- Pobierz: https://taskfile.dev/#/installation
  
- W folderze z kodem wykonaj komendy:
  
  Pierwsze uruchomienie:
    1. task build-images
    2. task up
    3. task install
    4. task db-reset
  
  Każde następne:
    1. task up
   
- Wyłączenie i usunięcie kontenerów: task down

- W przeglądarce połącz się z https://localhost
